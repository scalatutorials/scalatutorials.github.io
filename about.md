Scala Tutorials is an open-source collection of interactive Scala lessons that run
entirely in your browser. It was created by [Eran Medan](https://github.com/eranation)
in 2013 and modernized in 2026; the tour now targets **Scala 3**.

### How the code runs

The live editor on tour pages is [Scastie](https://scastie.scala-lang.org), the Scala
Center's official online playground. Your code is compiled and executed on Scastie's
servers: nothing to install, no account needed.

Historically the site ran on [ScalaKata](https://github.com/MasseGuillaume/ScalaKata2)
(and Codebrew before it) by Guillaume Massé and friends. Guillaume went on to create
Scastie, so the site's Run button has a fine pedigree.

### Contributing

Spotted an error or want to add a lesson? The whole site is on
[GitHub](https://github.com/scalatutorials/scalatutorials.github.io); issues and pull
requests welcome.
